# Feature Issue Template

## Title format

As a <persona>, I want <goal> [so that <benefit>]

## Label

enhancement

## Body sections

### Summary

Brief description of the feature request.

### Current State

Describe the current behavior or lack of functionality.

### Desired State

Describe the expected behavior after the feature is implemented.

### Added Value

Explain the benefit this feature brings to users or the project.

### Technical Notes (optional)

Any technical considerations, constraints, or implementation hints.
