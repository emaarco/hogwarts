# Bug Report Template

## Title format

<Component/Area>: <short description of the broken behavior>

## Label

bug

## Body sections

### Summary

Brief description of the bug.

### Steps to Reproduce

1. Step one
2. Step two
3. ...

### Current Behavior

Describe what currently happens (the bug).

### Expected Behavior

Describe what should happen instead.

### Technical Notes (optional)

Environment details, logs, screenshots, or any other relevant technical context.
